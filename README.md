# Flappy Bird – Version Web (HTML, CSS, JavaScript)

# Description
Ce projet est une **reproduction du jeu Flappy Bird** réalisée en **HTML, CSS et JavaScript**.  
Il a été développé dans un but **pédagogique**, afin de pratiquer la manipulation du DOM, la gestion des événements clavier/souris, les animations et la logique de jeu en JavaScript.

Ce projet n’est **pas une création originale**. Il s’agit d’une copie fidèle du jeu Flappy Bird, utilisée uniquement à des fins d’apprentissage.


# Fonctionnalités
- Contrôle de l’oiseau par clic ou touche du clavier  
- Génération dynamique des obstacles (tuyaux)
- Gestion des collisions
- Système de score
- Animation fluide en JavaScript
- Redémarrage du jeu après une défaite



# Technologies utilisées
- **HTML** : structure du jeu
- **CSS** : mise en forme et animations
- **JavaScript** : logique du jeu et interactions
